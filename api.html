GET/current
Currently playing level (first in queue, same as /queue?row=0).
response { "level": { "id": "144555909", "name": "make your ball ", "author": "colour4242", "difficulty": "Unrated", "requester": "malikhw47", "platform": "twitch", "message": "!replace 144555910 144555909", "description": "(No description provided)", "length": "Tiny", "large": false, "two_player": false, "timestamp": 0 } }
GET/add
Add a level by ID. Fetches metadata from GDBrowser. (incase youre doing a discord bot for requests from a channel...)
?id=676902
?platform=anything-like-discord
?message=wow
success { "ok": true }
error (not found) { "ok": false, "error": "level_not_found" }
POST/delete
Remove a level from the queue by ID.
body or query: id=12345
response { "ok": true }
POST/replace
Swap a level without changing its position from queue.
body or query: id=12345 new_id=785412396
response { "ok": true }
POST/banrequester
Blacklist the requester of a given level ID.
id=12345
response { "ok": true }
POST/banauthor
Blacklist the author of a given level ID.
id=12345
response { "ok": true }
POST/blacklistlevel
Blacklist a specific level by ID.
id=12345
response { "ok": true }
POST/clear
Clear the entire queue.
response { "ok": true }
POST/bantwitch
if there is a twitch session and moderation is on, this bans the requester from your Twitch chat.
id=12345
response { "ok": true }
GET/delete, /banrequester, /banauthor, /blacklistlevel, /clear
These endpoints also accept GET with the same parameters for convenience.
